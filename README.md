# Todo DevOps Lab

A step-by-step DevOps portfolio project.

## Goal

Start with a basic in-memory Todo application and gradually add:

- Backend API
- Frontend UI
- Docker
- Docker Compose
- GitHub Actions CI
- GitHub Actions CD
- PostgreSQL database
- Terraform IaC
- Kubernetes deployment
- Monitoring and logging

## Project Stages

1. Backend in-memory Todo API
2. Frontend with HTML, CSS, JavaScript
3. Dockerize backend and frontend
4. Run with Docker Compose
5. Add CI pipeline
6. Add CD pipeline
7. Add PostgreSQL
8. Add Terraform
9. Add Kubernetes
10. Add monitoring
